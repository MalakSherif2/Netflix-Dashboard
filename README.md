# 🎬 Netflix Content Dashboard

## 📊 Project Overview

This project analyzes Netflix content using an interactive Power BI dashboard.
It explores trends in movies and TV shows, including release years, ratings, and geographic distribution.

---

## 🛠 Tools & Technologies

* Power BI
* Data Cleaning
* Data Visualization

---

## 📁 Dataset

The dataset includes information about Netflix content such as:

* Title
* Type (Movie / TV Show)
* Director
* Country
* Release Year
* Rating

---

## 📌 Key Features

* KPI overview:

  * Total Movies
  * Total TV Shows
  * Total Directors
  * Total Countries

* Content distribution by:

  * Country
  * Type (Movie vs TV Show)
  * Release Year

* Top 5 Ratings analysis

* Interactive filters for better exploration

---

## 📈 Key Insights

* Movies dominate Netflix content compared to TV shows.

* The United States leads content production, followed by India and the United Kingdom.

* Netflix experienced rapid growth in content production starting from 2016, peaking around 2019–2020.

* Family-friendly ratings (TV-PG, TV-Y, TV-Y7) are among the most common.

* Despite growth in TV shows, movies remain the primary content type over time.

---

## 🎯 Key Takeaway

Netflix focuses heavily on scaling movie production globally, while maintaining a diverse content library across multiple countries and audience segments.
